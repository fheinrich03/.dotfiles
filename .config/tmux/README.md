# Tmux Config

## Prerequisites
- This config requires a [Nerd Font](https://www.nerdfonts.com/) to be installed and selected in your terminal emulator (e.g. ghostty)

## Installation
1. Follow the `Readme` in the Parent Folder
2. Install TPM first via `git clone https://github.com/tmux-plugins/tpm ~/.dotfiles/tmux/plugins/tpm/`
3. Delete or Backup your Tmux config file `~/.tmux.conf` if it exists
4. Reload Tmux the Config the first time by running `tmux source-file ~/.dotfiles/tmux/tmux.conf` in the terminal
5. and then run `<tmux-leader-key> + I` (tmux-leader-key should be `Ctrl + space` at this point)
