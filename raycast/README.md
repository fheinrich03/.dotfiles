# Raycast Config

- this config holds mostly the raycast scripts that I use

## Install Script Commands from this repository

To install new commands, follow these steps:

1. Open Raycast
2. Search for Extensions which should open the Extensions tab in the Raycast settings
4. Click the plus button (in the top right)
5. Click `Add Script Directory`
6. Select `~/.dotfiles/raycast/scripts/`
